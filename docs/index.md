# RDS App Documentation

The **RDS App** is an open-source web application for running Respondent-Driven Sampling (RDS) surveys — built for homelessness research and designed so that any city can adapt it for their own needs.

Originally developed in partnership with the University of Washington iSchool and the King County Regional Homelessness Authority (KCRHA) for the 2026 Point-in-Time Count, the app is now publicly available for other localities to fork, configure, and deploy.

## Is this for my city?

This project is a good fit if you:

- [x] Run Point-in-Time counts or other RDS-based surveys of unsheltered populations
- [x] Have (or can set up) a MongoDB instance and Twilio account
- [x] Have a developer who can fork a TypeScript/Node.js repo and configure environment variables
- [x] Want a free, self-hosted, open-source alternative to commercial survey tools

It may not be the right fit if you need a no-code solution, have strict on-premise data requirements, or need real-time population estimates built into the UI.

## Project status

The RDS App was used in the **2026 King County Unsheltered Point-in-Time Count** to collect surveys from approximately 3,700 unsheltered individuals. The codebase is actively maintained and open to community contributions.
